# 財富地圖 — 網站規劃文件

> 版本：v1.1　　最後更新：2026-05-30

---

## 一、專案定位

**財富地圖** 是一個個人理財管理網站，核心目標是讓使用者能夠：

1. 每月記錄資產快照（現金、股票、ETF）
2. 追蹤被動收入的成長軌跡
3. 視覺化呈現距離財務自由還有多遠

### 設計哲學

參考 **股癌（Gooaye）** 的理財理念：長期持有 ETF、建立被動收入、以「財務自由」為終極目標。
核心公式採用 **4% 法則（25 倍法則）**：財務自由目標 = 月支出 × 12 × 25。

> 工具只是工具，重要的是讓數字說話，讓進度可見，讓行動有方向。

## 三、響應式設計（RWD）方針

網站同時支援桌機與手機，以 **Mobile First** 原則設計，再擴展至桌機版。

| 斷點 | 裝置 | 導航方式 |
|------|------|---------|
| `xs` ~ `sm`（< 600px） | 手機 | 底部固定導航列 |
| `md`（≥ 900px） | 平板 / 桌機 | 左側側邊欄導航 |

### 桌機版佈局原則

- 左側：固定側邊欄（顯示 Logo + 4 個頁面連結）
- 右側：主要內容區，最大寬度 `1200px`，置中
- 資訊卡片以 Grid 排列（手機 1 欄 → 桌機 2~3 欄）
- 圖表在桌機版顯示更多資訊（可互動 tooltip）

## 四、幣別處理

### 支援幣別

| 代碼 | 名稱 | 主要用途 |
|------|------|---------|
| NTD | 新台幣 | 主幣別、顯示基準 |
| USD | 美元 | 美股 ETF |
| JPY | 日幣 | 現金存款（日圓帳戶） |

### 換算規則

- 所有資產統一換算為 **NTD** 計算總資產
- 匯率由使用者在設定頁面手動輸入，每月可更新
- 未來版本：接 ExchangeRate API 自動取得當日匯率

### 顯示規則

- 各幣別資產原始金額以原幣顯示（如：USD 3,000）
- 換算後 NTD 以淺色副標顯示（如：≈ NT$96,000）
- 總資產、圖表、計算公式一律使用 NTD

## 五、頁面規格

導航結構：**總覽 → 資產 → 現金流 → 目標**

---

### 頁面一：總覽

> 每次開啟的起點，30 秒掌握當前財務狀況。

**① 本月總資產**
- 顯示最新一筆紀錄的月份與總資產（NTD）
- 與上月差額（金額 + 百分比），正值綠色、負值紅色
- 歷史總資產走勢折線圖（視覺用）

**② 兩個關鍵指標（並排）**
- 左：本月被動收入 + 副標「預估股息 / 月」
- 右：本月儲蓄率（%）+ 副標本月淨現金流金額

**③ 資產組成概覽**
- 三行：銀行存款 / 台股 ETF / 美股 ETF
- 每行顯示金額（NTD）及與上月變化百分比

**④ 更新入口**
- 主要 CTA 按鈕：進入本月資料輸入流程

**⑤ 本月備註（若有）**

**空狀態**：尚無資料時顯示引導提示，進入第一筆資料輸入。

---

### 頁面二：資產

> 查看所有歷史紀錄，支援展開明細、編輯與刪除。

**① 最新總資產摘要**
- 最新月份總資產 + 歷史走勢折線圖（標示最早與最新月份）
- 「新增紀錄」按鈕

**② 歷史紀錄列表**
- 顯示共幾筆紀錄
- 每筆預設收合：月份、總資產、與上筆變化百分比
- 最新一筆加「最新」標記；有備註加備註標記
- 每筆有「編輯」與「刪除」操作

**展開單筆紀錄後顯示：**

| 區塊 | 內容 |
|------|------|
| 銀行存款 | 帳戶名稱、幣別、餘額（原幣 + NTD換算）、固定帳戶標記、小計 |
| 台股 ETF | 股票名稱、股數、股價（NTD）、市值、小計 |
| 美股 ETF | 股票名稱、股數、股價（USD）、匯率、市值（NTD）、小計 |
| 被動收入 | 本月實際收到金額 + 預估月股息 |
| 備註 | 若有則顯示 |

**編輯 / 刪除：**
- 「編輯」→ 開啟輸入流程並帶入原有資料
- 「刪除」→ 確認提示後才執行

---

### 頁面三：現金流

> 追蹤每月錢的流向，以及被動收入的成長趨勢。

**① 本月淨現金流**
- 最新月份淨現金流（正綠負紅）
- 拆解：存款變動（上月→本月）+ 本月被動收入

**② 被動收入概況（四個數字）**
- 本月實際收到
- 近 3 個月平均
- 預估股息 / 月（依持股自動計算）
- 年度累計

**③ 被動收入趨勢圖**
- 柱狀圖，每月被動收入，最新一筆以主色標示

**④ 存款走勢圖**
- 折線圖，每月銀行存款餘額變化

**資料不足提示：**少於 2 筆紀錄時，顯示提示說明。

---

### 頁面四：目標

> 設定財務目標，追蹤進度，預測達成時間。

**① 財務自由進度**
- 目標金額 = 月支出 × 12 × 25
- 目前進度百分比 + 進度條
- 目前總資產 vs 目標金額
- 月支出設定值、預估股息 / 月

**② 預估達成年份（三情境）**

| 情境 | 假設年報酬率 |
|------|------------|
| 保守 | 5% |
| 中性 | 8% |
| 樂觀 | 12% |

計算基準：以近期平均月增金額為投入推算，僅供參考。

**③ 個人目標清單**
- 名稱、目標金額、截止日期（選填）、進度（目前總資產 / 目標金額）、進度條
- 支援新增 / 編輯 / 刪除

---

### 輸入流程（新增 / 編輯紀錄）

**設計原則：存款人工輸入，股票系統自動計算**

| 數據來源 | 類型 |
|---------|------|
| 銀行存款餘額 | 手動輸入（最精確） |
| 持股股數 + 成交均價 | 手動設定（設定頁面一次設好，之後不變） |
| 股票目前市價 | **自動抓取**（TWSE API），也可手動調整 |
| 現值 / 損益 / 被動收入估算 | 系統自動計算 |

**步驟一：選擇本次要更新的內容**

新增紀錄後，先選擇本次更新範圍（可多選）：
- ☑ 銀行存款
- ☑ 股票市值

→ 只勾銀行：跳過股票步驟（適合月中確認存款但不看盤）
→ 只勾股票：跳過存款步驟（適合快速更新股市狀況）

**步驟二：存款帳戶**（若勾選）
- 選擇月份
- 活動帳戶：輸入當月餘額（支援 NTD / USD / JPY）
- 固定帳戶：自動帶入上月金額，可覆蓋修改

**步驟三：股票市值**（若勾選）
- 台股：系統自動呼叫 TWSE API 抓取即時股價，點「自動抓取」按鈕更新
- 美股：手動輸入股價（USD），搭配設定的匯率換算
- 每支股票顯示：股數 × 股價 = 市值（NTD）
- 股價來源：TWSE API > 上次紀錄的市價 > 空白

**步驟四：確認儲存**
- 顯示本次紀錄摘要（銀行 / 台股 / 美股 小計）
- 自動顯示預估月股息
- 輸入本月實際收到的被動收入
- 輸入備註（選填）

> **股票自動抓取技術說明**
> - 台股：呼叫 TWSE 即時行情 API（`mis.twse.com.tw`），CORS 問題透過 Vercel Edge Function 代理（dev 階段暫用 CORS proxy）
> - 美股：v1 手動，v2 串接 Yahoo Finance
> - 非交易時段：顯示最近一筆成交價

---

### 設定頁面（基礎設定）

- 帳戶管理：新增 / 編輯 / 刪除銀行帳戶（設定固定 / 活動）
- 持股管理：新增 / 編輯 / 刪除持股，支援兩種模式：
  - **手動填寫**：逐欄輸入股票資料
  - **截圖辨識**：上傳券商 App 截圖 → Claude Vision 自動解析 → 填入表單 → 確認後整筆覆蓋更新
  - 每次上傳截圖即代表最新狀態，直接覆蓋同代號股票（不新增一筆）
- 匯率設定：手動輸入 USD → NTD、JPY → NTD 匯率
- 財務目標設定：月支出金額（用於財務自由計算）
- 帳號管理：顯示登入帳號、登出

```
users/
  {userId}/
    settings/
      profile:
        - monthlyExpense: number        // 月支出（NTD）
        - displayCurrency: string       // 'NTD'（預設）

      exchangeRates:
        - USD_NTD: number               // 美元對台幣匯率
        - JPY_NTD: number               // 日幣對台幣匯率
        - updatedAt: timestamp

    accounts/                           // 銀行帳戶（設定，非月資料）
      {accountId}:
        - name: string                  // 帳戶名稱
        - currency: 'NTD' | 'USD' | 'JPY'
        - isFixed: boolean              // 固定帳戶（餘額不常變動）
        - order: number                 // 排列順序

    holdings/                           // 持股設定（每次截圖覆蓋更新）
      {holdingId}:
        - name: string                  // 股票名稱（如：元大台灣50）
        - ticker: string                // 代碼（如：0050）
        - market: 'TW' | 'US'
        - shares: number                // 總股數
        - avgCost: number               // 成交均價
        - currentPrice: number          // 最新市價（截圖更新）
        - totalCost: number             // 成本 = avgCost × shares
        - currentValue: number          // 現值 = currentPrice × shares
        - totalDividendReceived: number // 累積已領配息金額
        // 以下手動設定（截圖無法取得）
        - dividendPerShare: number      // 每股配息（最近一期，查公告填入）
        - dividendFrequency: number     // 每年配息次數（1=年配、2=半年、4=季配）
        - currency: 'TWD' | 'USD'
        - order: number
        - updatedAt: string             // 最後更新日期（YYYY-MM-DD）

    records/                            // 每月快照
      {recordId}:                       // 建議格式：YYYY-MM
        - month: string                 // '2026-05'
        - createdAt: timestamp
        - updatedAt: timestamp
        - note: string                  // 備註

        - bankSnapshots: [              // 各帳戶快照
            { accountId, name, currency, balance, balanceNTD }
          ]
        - bankTotal: number             // 銀行存款總計（NTD）

        - stockSnapshots: [             // 各持股快照
            { holdingId, name, ticker, market,
              shares, price, currency, marketValueNTD }
          ]
        - twStockTotal: number          // 台股小計（NTD）
        - usStockTotal: number          // 美股小計（NTD）

        - totalAssets: number           // 當月總資產（NTD）

        - passiveIncome: number         // 本月實際被動收入（NTD）
        - estimatedMonthlyDividend: number // 預估月股息（NTD，自動計算）

        - exchangeRates:               // 當月使用匯率（存入快照）
            { USD_NTD, JPY_NTD }

    goals/                             // 個人目標
      {goalId}:
        - name: string
        - targetAmount: number          // 目標金額（NTD）
        - deadline: timestamp | null
        - createdAt: timestamp
```

## 七、財務計算公式

### 財務自由目標

```
財務自由目標 = 月支出 × 12 × 25
財務自由進度 = 當前總資產 / 財務自由目標 × 100%
```

### 預估達成年份

使用複利終值反推：

```
目標金額 = 當前資產 × (1 + r)^n + 每月新增 × [(1 + r)^n - 1] / r

其中：
  r = 年報酬率 / 12（月報酬率）
  每月新增 = 近 6 個月平均月資產增加額
  n = 月數，解出 n 後換算為年份
```

### 預估配息（改為每股配息法）

```
年化配息 = 每股配息 × 持股股數 × 配息次數（dividendFrequency）
預估月股息 = 年化配息 / 12

// 範例：0050 每股配息 3.50 元，半年配（2次），持有 433 股
// 年化配息 = 3.50 × 433 × 2 = 3,031 元
// 預估月股息 = 3,031 / 12 ≈ 253 元
```

> 改用「每股配息 × 股數」的原因：年殖利率會隨市價波動，每股配息是除息前公告的固定金額，計算結果更穩定且貼近實際。

### 淨現金流

```
淨現金流 = (本月銀行存款 - 上月銀行存款) + 本月被動收入
儲蓄率 = 淨現金流 / (淨現金流 + 本月支出) × 100%
// 備註：本月支出暫時以「設定的月支出」代替，未來加入支出模組後更準確
```
