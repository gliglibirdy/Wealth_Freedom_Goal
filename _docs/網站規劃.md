# 財富地圖 — 網站規劃文件

> 版本：v1.2　　最後更新：2026-05-30

---

## 一、專案定位

**財富地圖** 是一個個人理財管理網站，核心目標是讓使用者能夠：

1. 每月記錄資產快照（現金、股票、ETF）
2. 追蹤被動收入的成長軌跡
3. 視覺化呈現距離財務自由還有多遠

### 設計哲學

參考 **股癌（Gooaye）** 的理財理念：長期持有 ETF、建立被動收入、以「財務自由」為終極目標。
核心公式採用 **4% 法則（25 倍法則）**：財務自由目標 = 月支出 × 12 × 25。

> 工具只是工具，重要的是讓數字說話，讓進度可見，讓行動有方向。

## 四、幣別處理

### 支援幣別

| 代碼 | 名稱 | 主要用途 |
|------|------|---------|
| NTD | 新台幣 | 主幣別、顯示基準 |
| USD | 美元 | 美股 ETF |
| JPY | 日幣 | 現金存款（日圓帳戶） |

### 換算規則

- 所有資產統一換算為 **NTD** 計算總資產
- 匯率由使用者在設定頁面手動輸入，每月可更新
- 未來版本：接 ExchangeRate API 自動取得當日匯率

### 顯示規則

- 各幣別資產原始金額以原幣顯示（如：USD 3,000）
- 換算後 NTD 以淺色副標顯示（如：≈ NT$96,000）
- 總資產、圖表、計算公式一律使用 NTD

### 頁面一：總覽

> 每次開啟的起點，30 秒掌握當前財務狀況。

**① 本月總資產**
- 顯示最新一筆紀錄的月份與總資產（NTD）
- 與上月差額（金額 + 百分比），正值綠色、負值紅色
- 歷史總資產走勢折線圖（視覺用）

**② 兩個關鍵指標（並排）**
- 左：本月預期配息（依配息月份推算）+ 副標「年預期 NT$ XXXX」
- 右：本月儲蓄率（%）+ 副標本月淨現金流金額

**③ 資產組成概覽**
- 三行：銀行存款 / 台股 ETF / 美股 ETF
- 每行顯示金額（NTD）及與上月變化百分比
- 台股市值旁顯示「最後更新：HH:MM」（FinMind 資料時效）

**④ 更新入口**
- 主要 CTA 按鈕：進入本月資料輸入流程

**⑤ 本月備註（若有）**

**空狀態**：尚無資料時顯示引導提示，進入第一筆資料輸入。

> 查看所有歷史紀錄，支援展開明細、編輯與刪除。

**① 最新總資產摘要**
- 最新月份總資產 + 歷史走勢折線圖（標示最早與最新月份）
- 「新增紀錄」按鈕

**② 歷史紀錄列表**

- 最新一筆加「最新」標記；有備註加備註標記
- 每筆有「編輯」與「刪除」操作

**展開單筆紀錄後顯示：**

| 區塊 | 內容 |
|------|------|

- 「編輯」→ 開啟輸入流程並帶入原有資料
- 「刪除」→ 確認提示後才執行

### 頁面三：現金流

> 追蹤每月錢的流向，以及被動收入的成長趨勢。

**① 本月淨現金流**
- 最新月份淨現金流（正綠負紅）
- 拆解：存款變動（上月→本月）+ 本月被動收入

**② 被動收入概況（五個數字）**
- 本月實際收到
- 近 3 個月平均
- 年預期配息 ÷ 12（月均，依持股 × FinMind 近 3 次配息平均計算）；備註：季節性波動，非每月實領
- 本月預期配息（依配息月份顯示，非配息月顯示 NT$0 或不配息說明）
- 年度累計

**③ 被動收入趨勢圖**
- 柱狀圖，每月被動收入，最新一筆以主色標示

**④ 存款走勢圖**
- 折線圖，每月銀行存款餘額變化

**資料不足提示：**少於 2 筆紀錄時，顯示提示說明。

### 頁面四：目標

> 設定財務目標，追蹤進度，預測達成時間。

**① 財務自由進度**
- 目標金額 = 月支出 × 12 × 25
- 目前進度百分比 + 進度條
- 目前總資產 vs 目標金額
- 月支出設定值、預估股息 / 月

**② 預估達成年份（三情境）**

| 情境 | 假設年報酬率 |
|------|------------|
| 保守 | 5% |
| 中性 | 8% |
| 樂觀 | 12% |

計算基準：以近期平均月增金額為投入推算，僅供參考。

**③ 個人目標清單**
- 名稱、目標金額、截止日期（選填）、進度（目前總資產 / 目標金額）、進度條
- 支援新增 / 編輯 / 刪除

### 輸入流程（新增 / 編輯紀錄）

**設計原則：存款人工輸入，台股市值系統自動計算，步驟二從「輸入」變為「確認」**

| 數據來源 | 類型 |
|---------|------|
| 銀行存款餘額 | 手動輸入（最精確） |
| 持股股數 | 手動設定（設定頁面一次設好，持股數變動時更新） |
| 台股目前股價 | **自動抓取**（FinMind API），可手動覆蓋當次數字 |
| 美股股價 | 手動輸入（USD） |
| 市值 / 配息預估 | 系統自動計算 |

**步驟一：選擇本次要更新的內容**

新增紀錄後，先選擇本次更新範圍：
- ☑ 銀行存款
- ☑ 股票市值

→ 只選銀行：跳過股票步驟，股票市值沿用上次紀錄
→ 只選股票：跳過存款步驟，存款沿用上次紀錄

**步驟二：存款帳戶**（若選擇）
- 選擇月份
- 活動帳戶：輸入當月餘額（支援 NTD / USD / JPY）
- 固定帳戶：自動帶入上月金額，可覆蓋修改

**步驟三：股票市值確認**（若選擇）
- **台股：** 顯示「系統已自動從 FinMind 取得最新股價」，列出每支股票的當前市值供確認
  - 每支股票顯示：代碼、名稱、持股數、FinMind 收盤價、計算市值
  - 若使用者認為股價有誤，可點擊覆蓋單筆股價（儲存時標注「手動輸入」）
- **美股：** 維持手動輸入股價（USD），搭配設定的匯率換算
- 整體流程從「輸入股價」改為「確認市值」，大幅縮短操作時間

**步驟四：確認儲存**
- 顯示本次紀錄摘要（銀行 / 台股 / 美股 小計）
- 輸入本月實際收到的被動收入（對照配息實際入帳）
- 輸入備註（選填）
- 移除舊版「自動顯示預估月股息」區塊（改由頁面常駐顯示）

> **台股自動抓取技術說明**
> - 來源：FinMind API，`TaiwanStockPrice` 資料集
> - 時機：每次開啟 App 自動更新一次；非交易時間顯示最後一個交易日收盤價
> - CORS：FinMind API 支援瀏覽器直接呼叫，無需 proxy

### 設定頁面（基礎設定）

- 帳戶管理：新增 / 編輯 / 刪除銀行帳戶（設定固定 / 活動）
- 持股管理：新增 / 編輯 / 刪除持股，支援兩種模式：
  - **手動填寫（台股）**：輸入股票代碼 + 持股數（移除股價與殖利率欄位，改由 FinMind 自動取得）

  - 頁面上方提供「重新整理資料」按鈕，手動觸發 FinMind 更新（股價 + 配息）
- 匯率設定：手動輸入 USD → NTD、JPY → NTD 匯率
- 財務目標設定：月支出金額（用於財務自由計算）
- 帳號管理：登出

## 七、財務計算公式

### 財務自由目標

```
財務自由目標 = 月支出 × 12 × 25
財務自由進度 = 當前總資產 / 財務自由目標 × 100%
```

### 預估達成年份

使用複利終值反推：

```
目標金額 = 當前資產 × (1 + r)^n + 每月新增 × [(1 + r)^n - 1] / r

其中：
  r = 年報酬率 / 12（月報酬率）
  每月新增 = 近 6 個月平均月資產增加額
  n = 月數，解出 n 後換算為年份
```

### 預估配息（FinMind 近 3 次平均法）

```
// 台股（FinMind 自動計算）
每支台股預期年配息 = avg(近 3 次 dividendPerShare) × 持股數

// 美股（手動輸入）
每支美股預期年配息 = 股價（USD） × annualYieldPct / 100 × 持股數 × 匯率

// 合計
年預期配息 = 所有台股預期年配息加總 + 所有美股預期年配息加總
月均預期配息 = 年預期配息 / 12

// 本月預期配息（依配息月份）
本月預期配息 = 本月為配息月的股票預期配息加總
```

**各 ETF 配息月份對照：**

| ETF 類型 | 代表 | 約配息月份 |
|---------|------|----------|
| 台股半年配 | 0050、006208 | 1月、7月 |
| 台股季配 | 00878、00713 | 2、5、8、11月 |
| 美股季配 | VT、SCHD | 3、6、9、12月 |

> 改用「FinMind 近 3 次配息平均 × 股數」的原因：年殖利率隨市價波動，歷史配息金額是除息前公告的固定數字，計算結果更穩定，且不受市值漲跌影響。

### 淨現金流

```
淨現金流 = (本月銀行存款 - 上月銀行存款) + 本月被動收入
儲蓄率 = 淨現金流 / (淨現金流 + 本月支出) × 100%
// 備註：本月支出暫時以「設定的月支出」代替，未來加入支出模組後更準確
```

### API 1：取得台股最新股價

**用途：** 計算台股持倉市值

```js
// src/services/finmind.js
const BASE = 'https://api.finmindtrade.com/api/v4/data'
const TOKEN = import.meta.env.VITE_FINMIND_TOKEN

// 取得單一股票最新收盤價（取近 5 個交易日，用最新一筆）
export async function fetchTWStockPrice(ticker) {
  const today = new Date().toISOString().slice(0, 10)
  const startDate = new Date(Date.now() - 7 * 86400000).toISOString().slice(0, 10)

  const url = new URL(`${BASE}`)
  url.searchParams.set('dataset', 'TaiwanStockPrice')
  url.searchParams.set('data_id', ticker)          // 如 '0050'
  url.searchParams.set('start_date', startDate)
  url.searchParams.set('end_date', today)
  url.searchParams.set('token', TOKEN)

  const res = await fetch(url)
  const json = await res.json()
  const rows = json.data || []
  if (!rows.length) return null

  // 依日期降序取最新收盤價
  rows.sort((a, b) => b.date.localeCompare(a.date))
  return { price: rows[0].close, date: rows[0].date }
}

// 批次取得多支台股（逐一呼叫，避免超出頻率限制）
export async function fetchAllTWPrices(tickers) {
  const results = {}
  for (const ticker of tickers) {
    try {
      results[ticker] = await fetchTWStockPrice(ticker)
    } catch {
      results[ticker] = null
    }
    await new Promise(r => setTimeout(r, 300)) // 限流：每次間隔 300ms
  }
  return results
}
```

**回傳範例：**
```json
{
  "data": [
    { "date": "2026-05-29", "stock_id": "0050", "open": 180.0, "close": 181.5, "volume": 12345 }
  ]
}
```

### API 2：取得台股配息紀錄

**用途：** 計算預期年配息（近 3 次平均每單位配息）

```js
// 取得近 3 年配息紀錄
export async function fetchTWStockDividend(ticker) {
  const threeYearsAgo = new Date(Date.now() - 3 * 365 * 86400000).toISOString().slice(0, 10)
  const today = new Date().toISOString().slice(0, 10)

  const url = new URL(`${BASE}`)
  url.searchParams.set('dataset', 'TaiwanStockDividend')
  url.searchParams.set('data_id', ticker)
  url.searchParams.set('start_date', threeYearsAgo)
  url.searchParams.set('end_date', today)
  url.searchParams.set('token', TOKEN)

  const res = await fetch(url)
  const json = await res.json()
  const rows = json.data || []

  // 依除息日降序，取近 3 次現金股息
  const cashDividends = rows
    .filter(r => r.CashDividend > 0)
    .sort((a, b) => b.date.localeCompare(a.date))
    .slice(0, 3)

  if (!cashDividends.length) return { avgDividendPerShare: 0, records: [] }

  const avg = cashDividends.reduce((s, r) => s + r.CashDividend, 0) / cashDividends.length
  return {
    avgDividendPerShare: avg,
    records: cashDividends.map(r => ({ date: r.date, dividendPerShare: r.CashDividend })),
  }
}
```

**回傳欄位說明（`TaiwanStockDividend`）：**

| 欄位 | 說明 |
|------|------|
| `date` | 除息基準日 |
| `CashDividend` | 現金股息（每股，NTD） |
| `StockDividend` | 股票股息（可忽略） |
| `CashDividendPaymentDate` | 配息入帳日期 |

### 更新時機與快取策略

```js
// src/services/finmind.js

// 檢查是否需要更新股價（每次開啟 App 自動更新）
export function isPriceStale(updatedAt) {
  if (!updatedAt) return true
  const updated = new Date(updatedAt)
  const now = new Date()
  // 超過 8 小時視為過期（非交易日不影響，仍顯示最後收盤價）
  return (now - updated) > 8 * 60 * 60 * 1000
}

// 檢查是否需要更新配息（每週一次）
export function isDividendStale(updatedAt) {
  if (!updatedAt) return true
  const updated = new Date(updatedAt)
  return (now - updated) > 7 * 24 * 60 * 60 * 1000
}
```

**儲存位置：** 更新後寫回 Firestore `holdings/{id}` 的 `finmindPrice`、`finmindPriceUpdatedAt`、`finmindDividends`、`finmindDividendUpdatedAt` 欄位。

### 計算預期配息（整合後）

```js
// 台股
function calcTWAnnualDividend(holding) {
  const avgDiv = holding.finmindDividends?.[0]?.dividendPerShare  // 或平均值
    ?? holding.avgDividendPerShare ?? 0
  return avgDiv * holding.shares
  // 注意：FinMind CashDividend 已是「每次」發放金額，非「每年」
  // 若一年配 2 次，近 3 次平均後直接 × 股數 = 近一次的單次配息額
  // 年配息 = avg(近 3 次 CashDividend) × shares × 年配息次數（需另外判斷）
}

// 判斷配息頻率（簡易版：依近 3 次配息日期間距）
function inferFrequency(dividendRecords) {
  if (dividendRecords.length < 2) return 2   // 預設半年配
  const intervals = dividendRecords.slice(0, -1).map((r, i) => {
    const a = new Date(r.date), b = new Date(dividendRecords[i + 1].date)
    return Math.abs(a - b) / (30 * 24 * 60 * 60 * 1000)  // 月數
  })
  const avgInterval = intervals.reduce((s, v) => s + v, 0) / intervals.length
  if (avgInterval < 4) return 4    // 季配
  if (avgInterval < 8) return 2    // 半年配
  return 1                          // 年配
}
```

### 開發注意事項

1. **免費帳戶限制**：每分鐘 30 次。批次更新多支股票時，每次請求間隔 ≥ 300ms。
2. **Token 安全性**：`VITE_FINMIND_TOKEN` 會暴露在前端 JS，這是個人使用 App 的可接受風險（與 Anthropic Key 相同情況）。
3. **非交易時間**：FinMind 不提供盤中即時價格（免費版），只有收盤價。`start_date` 設為 7 天前可確保拿到最近一個交易日的收盤價。
4. **股票代碼格式**：FinMind 使用純數字代碼（如 `"0050"`），不需加 `.TW` 後綴。
5. **配息資料延遲**：FinMind 配息資料通常在除息日後數日更新，若剛除息可能尚未反映。
